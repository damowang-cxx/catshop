# 设计更新说明 - 卡其色系玩偶商店风格

## 设计理念

将前端设计调整为适合销售玩偶的温暖、稳定风格，使用卡其色系（khaki/beige/tan）作为主色调，营造温馨、专业的购物体验。

## 颜色方案

### 主色调
- **主色**: `amber-700` / `amber-800` (深卡其色/棕色) - 用于按钮、强调元素
- **浅色**: `stone-50` / `amber-50` (米白色/浅卡其色) - 用于背景
- **中性色**: `stone-*` 系列 - 用于文字、边框、背景

### 具体颜色映射

| 原色 | 新色 | 用途 |
|------|------|------|
| `blue-600` | `amber-700` | 按钮、价格标签、强调色 |
| `neutral-50` | `stone-50` | 页面背景 |
| `neutral-200` | `stone-200` | 边框 |
| `neutral-500` | `stone-600` | 次要文字 |
| `neutral-900` | `stone-900` | 深色模式背景 |
| `black` | `stone-900` | 主要文字 |
| `white` | `stone-50` / `stone-100` | 浅色背景/文字 |

## 设计特点

### 1. 温暖色调
- 使用卡其色、米色、棕色等暖色调
- 营造温馨、舒适的购物氛围
- 适合展示玩偶等可爱商品

### 2. 稳定专业
- 使用 `stone` 系列中性色作为基础
- 保持足够的对比度确保可读性
- 统一的颜色系统，视觉一致

### 3. 细节优化
- 所有按钮添加 `transition-colors` 过渡效果
- 悬停状态使用更深的卡其色 (`amber-800`)
- 保持深色模式支持

## 视觉效果

### 主要变化
1. **按钮**: 从蓝色 (`blue-600`) 改为深卡其色 (`amber-700`)
2. **背景**: 从纯白/纯黑改为温暖的米白色 (`stone-50`) / 深棕色 (`stone-900`)
3. **文字**: 从纯黑/纯白改为深棕色 (`stone-900`) / 浅米色 (`stone-100`)
4. **边框**: 从灰色改为浅卡其色 (`stone-200`)

## 技术细节

所有颜色更改都使用了 Tailwind CSS 的颜色类：
- `amber-*`: 卡其色/琥珀色系列
- `stone-*`: 石灰色系列（中性色）
- 保持了响应式设计和深色模式支持
